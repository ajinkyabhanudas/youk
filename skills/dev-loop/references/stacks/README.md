# Stack References

This directory contains framework-specific reference files generated automatically
by the dev-loop skill when a new stack is encountered.

Each file is named after the framework it covers, e.g.:
- react.md
- nextjs.md
- vue.md
- django.md
- svelte.md

Files here are generated during the UNDERSTAND phase when a stack gap is detected
and the user confirms generation. They are loaded alongside the base reference
files during all relevant phases.
